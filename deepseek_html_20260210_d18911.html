Clique no botão abaixo para salvar a imagem capturada no seu dispositivo.

Importante

Por questões de segurança, os navegadores não permitem que JavaScript salve arquivos diretamente em pastas do sistema. Esta aplicação permite que você salve a imagem na galeria ou pasta de downloads do seu dispositivo usando as funcionalidades do navegador.

Como usar:

  1. Clique em "Iniciar Câmera" para ativar a câmera do seu dispositivo
  2. Quando a câmera estiver ativa, clique em "Capturar Imagem" para tirar uma foto
  3. A imagem capturada será exibida na tela
  4. Clique em "Salvar no Dispositivo" para baixar a imagem para seu celular
  5. Você também pode compartilhar a imagem usando o botão "Compartilhar"
